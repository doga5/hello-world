# 2. Styling Text

`* *` or `_ _` :      *italic text*

`** **` or ` __ __` : __bold text__

`~~ ~~` :            ~~strikethrough text~~ 

`*** ***` :        ***bold and italic text***

`**` and `_` :    **to italicize a _word_ in a bold sentence**

`>`: 
> to quote a text
